
# Mariusz Najwer - Sudoku
[https://najwer23.github.io/sudoku/](https://najwer23.github.io/sudoku)

## Self Update
```
npm i -g npm-check-updates
ncu -u
npm install
```

## Installation
```sh
$ yarn install
```

## Development
```sh
$ yarn start
```

## Production
```sh
$ yarn build
```

## Deploy to gh-pages
```sh
$ yarn deploy
```
